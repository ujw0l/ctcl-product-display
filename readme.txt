Gutenberg block to add product display in page or post

== Description ==

The gutenberg block for CT Commerce lite to display feature products or related product in page.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/ctcl-product-display` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Where do I use this block? =

This block can be used to display featured product in home page or related product in product page. 

= What is best use case? =

Use column block from built in blocks and add this block to each column.
